## 📖 Overview

**Context AI** is a context-aware personal productivity assistant that integrates with Google Workspace services and leverages a locally hosted Large Language Model to transform unstructured text into actionable tasks.

The system continuously gathers context from:

* 📅 Google Calendar Events
* 📧 Gmail Unread Messages
* ✅ User Created Tasks

and combines them with local AI-powered natural language processing using **Llama 3.2 running via Ollama**.

The goal is to provide a privacy-focused workspace orchestration engine capable of converting everyday thoughts, emails, and reminders into structured task management workflows.

# 🔄 Complete Data Flow

```text
1. User clicks Login

        ↓

2. React redirects user to

   http://localhost:8080/oauth2/authorization/google

        ↓

3. Spring Security handles OAuth2 request

        ↓

4. Google Authentication Success

        ↓

5. User information saved in MySQL

        ↓

6. JWT generated by backend

        ↓

7. React receives JWT

        ↓

8. JWT stored as:

   assistant_jwt

        ↓

9. Dashboard loads

        ↓

10. Polling starts every 10 seconds

        ↓

11. GET /api/tasks

12. GET /api/assistant/dashboard-context

        ↓

13. Backend retrieves:

    - Tasks
    - Calendar Events
    - Gmail Messages

        ↓

14. Data returned to React

        ↓

15. Workspace Dashboard Updated
```

# 🐞 Challenge Log & Solutions

## 1. OAuth2 Token Expiration

### Problem

Google access tokens expired after 60 minutes.

### Solution

Implemented:

```java
DefaultOAuth2AuthorizationRequestResolver
```

to inject:

```text
access_type=offline
prompt=consent
```

This enabled Google to issue a persistent refresh token.

---

## 2. Gmail Scope Misalignment

### Problem

Dashboard showed empty email widgets.

### Root Cause

Application had Draft scope while querying inbox messages.

### Solution

Changed scope to:

```text
https://www.googleapis.com/auth/gmail.readonly
```

and filtered:

```text
is:unread category:primary
```

---

## 3. Jackson Infinite Serialization Loop

### Problem

500 Internal Server Error during task updates.

### Solution

Added:

```java
@JsonIgnoreProperties({
 "hibernateLazyInitializer",
 "handler",
 "tasks",
 "tokens"
})
```

to break cyclic Hibernate relationships.

---

## 4. Calendar Synchronization Bug

### Problem

Dashboard showed outdated events.

### Solution

Added:

```java
.setTimeMin(
  new DateTime(System.currentTimeMillis())
)
.setOrderBy("startTime")
.setSingleEvents(true)
```

to fetch only future events.
